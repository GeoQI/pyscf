nao — Numerical Atomic Orbitals¶

The nao module is a simple and elegant python library to perform iiterative Time Dependent Density Functional Theory (TDDFT) using ground state data from differents programs,

  • Siesta
  • PYSCF
  • OpenMX
  • GPAW
