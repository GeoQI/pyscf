44.8. hfc — Hyperfine coupling tensor¶

To compute the hyperfine coupling tensor, one could follow the following example:

from pyscf import gto, scf, dft
from pyscf.prop import hfc
mol = gto.M(atom='''
            C 0 0 0
            N 0 0 1.1747
            ''',
            basis='ccpvdz', spin=1, charge=0, verbose=3)
mf = scf.UHF(mol).run()
gobj = hfc.uhf.HFC(mf).set(verbose=4)
gobj.para_soc2e = 'SSO+SOO'
gobj.so_eff_charge = False
gobj.kernel()

pyscf.prop.hfc.uhf.make_h1_soc(hfcobj, dm0)[source]¶

1-electron and 2-electron spin-orbit coupling integrals.

1-electron SOC integral is the imaginary part of [i sigma dot pV x p], ie [sigma dot pV x p].

Note sigma_z is considered in the SOC integrals (the (-) sign for beta-beta block is included in the integral). The factor 1/2 in the spin operator s=sigma/2 is not included.
